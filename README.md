# Book My Stay App

Incremental Core Java use cases for hotel booking system design.

## Prerequisite
- Java JDK installed (`javac`, `java` available in PATH)

## Compile and Run
Use these commands from the project root:

### UC1
```bash
javac UC1ApplicationStartup.java
java UC1ApplicationStartup
```

### UC2
```bash
javac UC2RoomInitialization.java
java UC2RoomInitialization
```

### UC3
```bash
javac UC3InventorySetup.java
java UC3InventorySetup
```

### UC4
```bash
javac UC4RoomAvailabilityView.java
java UC4RoomAvailabilityView
```

### UC5
```bash
javac UC5BookingRequestQueue.java
java UC5BookingRequestQueue
```

### UC6
```bash
javac UC6ReservationConfirmation.java
java UC6ReservationConfirmation
```

### UC7
```bash
javac UC7AddOnServiceSelection.java
java UC7AddOnServiceSelection
```

### UC8

### UC10
